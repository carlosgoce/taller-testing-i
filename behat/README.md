# Behat

## Prerequisitos

- PHP >= 5.3.3
- [Composer](https://getcomposer.org/)

## Instalación

```bash
composer install
```

## Ejecución

```bash
bin/behat
```

## Ayuda

Ver sintáxis gherkin

```bash
bin/behat --lang=es --story-syntax
```

- [Behat](http://behat.org/en/latest/guides.html)
